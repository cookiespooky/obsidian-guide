Эта заметка показывает базовый Markdown, который понимает Obsidian.  

## Мини-набор, который реально нужен каждый день

```
# Заголовок  
- список  
- [ ] задача  
[[ссылка]]  
> цитата  
`код`
```

Если вы запомните только это - уже достаточно. Можно вместо ручного ввода использовать готовые Markdown шаблоны, для этого в редакторе Obsidian-заметки нужно кликнуть правой кнопкой мыши и выбрать нужный элемент.
  
---  
## Весь синтаксис
### Заголовки  
  
Примеры (как писать):  
  
```md  
# Заголовок 1  
## Заголовок 2  
### Заголовок 3  
#### Заголовок 4
```

---
### Жирный, курсив, зачёркнутый

```
**жирный**  
*курсив*  
~~зачёркнутый~~
```
---
### Списки

Маркированный список:

```
- пункт  
- пункт  
  - подпункт
```

Нумерованный список:

```
1. пункт  
2. пункт  
3. пункт
```

Чекбоксы (задачи):

```
- [ ] задача  
- [x] выполнено
```

---
### Ссылки (Obsidian)

Ссылка на заметку:

`[[Название заметки]]`

Ссылка на заметку с другим текстом:

`[[Название заметки|Текст ссылки]]`

Ссылка на заголовок внутри заметки:

`[[Название заметки#Заголовок]]`

---
### Обычные ссылки (URL)

`https://example.com`

Markdown-ссылка:

`[текст](https://example.com)`

---
### Цитаты

```
> Это цитата  
> Вторая строка цитаты
```

---
### Разделитель

`---`

---
### Код

Инлайн-код:

`console.log("hi")`

Код-блок:

```js  
console.log("hi")  
```

---
### Таблицы

```
| Колонка | Колонка |  
|--------|---------|  
| текст  | текст   |  
| текст  | текст   |
```

---
### Изображения

Картинка из файла (если лежит в vault):

![[image.png]]

Картинка по ссылке:

```
![Картинка](https://example.com/image.png)
```
---
### Выделение текста

```
==подсветка==
```

---
### Комментарии (скрытый текст)

Если нужно оставить заметку себе и не показывать её в режиме чтения:

```
%% это комментарий, его не видно в preview %%
```
