# Полезные плагины  
  
Вот несколько плагинов, которые часто используют пользователи Obsidian.  
  
---  
  
## Calendar  
  
Добавляет календарь для навигации по **Daily Notes**.  
  
Очень удобен если вы ведёте ежедневные заметки.  
  
---  
  
## Templater  
  
Расширяет возможности шаблонов.  
  
Позволяет:  
  
- автоматически вставлять дату  
- генерировать структуру заметок  
- создавать динамические шаблоны  
  
---  
  
## Tasks  
  
Плагин для управления задачами внутри заметок.  
  
Позволяет создавать задачи и собирать их в списки.  
  
---  
  
## Dataview  
  
Один из самых мощных плагинов Obsidian.  
  
Позволяет строить динамические списки заметок.  
  
Например:  
  
- список проектов  
- список статей  
- список задач
