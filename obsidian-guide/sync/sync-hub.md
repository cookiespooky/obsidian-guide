# Синхронизация Obsidian между устройствами  
  
Obsidian хранит заметки как обычные **Markdown-файлы** на вашем устройстве.  
  
Это означает, что вы полностью владеете своими данными.  
Но это также означает, что синхронизацию между устройствами нужно настроить отдельно.  
  
Например, если вы используете:  
  
- компьютер  
- телефон  
- планшет  
  
вам понадобится способ **синхронизировать vault между ними**.  
  
Хорошая новость - вариантов несколько.  
  
---  
## Основные способы синхронизации  
  
Существует несколько распространённых способов синхронизации.

### 1. Obsidian Sync  
  
Официальный сервис синхронизации от разработчиков Obsidian.  
  
Плюсы:  
  
- простая настройка  
- работает на всех платформах  
- поддерживает историю версий  
  
Минусы:  
  
- платная подписка  
  
Подробнее:  
  
→ Синхронизация через [[obsidian-sync|Obsidian Sync]]
  
---  
### 2. WebDAV  
  
Синхронизация через сервер WebDAV.  
  
Этот вариант популярен среди тех, кто хочет **self-hosted решение**.  
  
Плюсы:  
  
- контроль над данными  
- можно использовать собственный сервер  
  
Минусы:  
  
- требуется настройка сервера  
  
Подробнее:  
  
→ [[webdav|Синхронизация через WebDAV]]  
  
---
### 3. Syncthing  

Syncthing — это система прямой синхронизации между устройствами (peer-to-peer).

Файлы передаются напрямую между вашими устройствами без облачного сервера.

Плюсы:

- полностью бесплатно  
- данные остаются только у вас  
- высокая скорость в локальной сети  

Минусы:

- устройства должны периодически быть онлайн  
- нет официального клиента для iOS  

Подробнее:

→ [[syncthing|Синхронизация через Syncthing]]

---
### 4. Git  
  
Некоторые пользователи синхронизируют vault через Git.  
  
Плюсы:  
  
- контроль версий  
- возможность резервного копирования  
  
Минусы:  
  
- требует базовых знаний Git  
- может быть неудобно на мобильных устройствах  
  
Подробнее:  
  
→ [[git|Синхронизация через Git]]
  
---  
### 5. S3  
  
Облачное хранилище, совместимое с S3.  
  
Например:  
  
- AWS S3  
- MinIO  
- другие S3-совместимые сервисы  
  
Плюсы:  
  
- масштабируемость  
- гибкость  
  
Минусы:  
  
- требует настройки  
  
Подробнее:  
  
→ [[s3|Синхронизация через S3]]

## Какой вариант выбрать

| метод         | бесплатно | iOS | сложность |
| ------------- | --------- | --- | --------- |
| Obsidian Sync | ❌         | ✅   | ⭐         |
| Syncthing     | ✅         | ⚠️  | ⭐⭐        |
| WebDAV        | ✅         | ✅   | ⭐⭐        |
| Git           | ✅         | ⚠️  | ⭐⭐⭐       |
| S3            | ⚠️        | ✅   | ⭐⭐⭐       |
  
Если вы только начинаете использовать Obsidian:  
  
- самый простой вариант — **Obsidian Sync**  
  
Если вы хотите больше контроля:  
  
- WebDAV  
- S3  
- Git
- Syncthing
  
Выбор зависит от:  
  
- ваших устройств  
- опыта работы с инфраструктурой  
- требований к безопасности
