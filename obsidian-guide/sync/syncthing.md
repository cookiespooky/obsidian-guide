# Синхронизация Obsidian через Syncthing

Syncthing — это система **прямой синхронизации устройств**.

Файлы передаются **напрямую между устройствами**, без облака и сторонних серверов.

Это означает:

- ваши данные остаются у вас
- синхронизация происходит напрямую
- нет ограничений облачных сервисов

Syncthing работает на:

- Windows
- macOS
- Linux
- Android

На iOS официального клиента нет.

---

# Когда выбирать Syncthing

Syncthing подходит, если:

- не хотите хранить заметки в облаке
- важна приватность
- хотите полностью бесплатную синхронизацию
- устройства часто находятся онлайн одновременно

---

# Как работает синхронизация

Syncthing соединяет устройства напрямую.

Ноутбук  
   ↓  
Syncthing  
   ↓  
Телефон  
   ↓  
Syncthing

Файлы передаются:

- напрямую между устройствами
- через зашифрованное соединение
- без промежуточного сервера

Каждое устройство хранит **полную копию vault**.

# Настройка

## Шаг 1. Установить Syncthing

Скачать можно здесь:

[https://syncthing.net](https://syncthing.net)

Установить на всех устройствах.

---

# Шаг 2. Запустить Syncthing

После запуска откроется веб-интерфейс.

Обычно он доступен по адресу:

http://127.0.0.1:8384

В нём происходит вся настройка.

---

# Шаг 3. Подключить устройства

Каждое устройство имеет **Device ID**.

Чтобы соединить устройства:

1. Нажать **Add Remote Device**
2. Вставить Device ID второго устройства
3. Подтвердить соединение

После этого устройства будут связаны.

---

# Шаг 4. Добавить папку vault

Нужно добавить папку Obsidian vault.

В Syncthing: **Add Folder**

Указать:

- Folder Path → путь к vault
- Folder Label → любое имя

Например: Obsidian Vault

Затем выбрать устройства, которые будут синхронизироваться.

---

# Шаг 5. Подтвердить синхронизацию

На втором устройстве появится запрос: **Accept shared folder**

После подтверждения начнётся синхронизация.

---

# Первая синхронизация

При первой синхронизации:

- все файлы vault копируются на второе устройство
- время зависит от размера vault

После этого синхронизация происходит мгновенно.

---

# Конфликты и подводные камни

### Одновременное редактирование

Если файл изменён на двух устройствах одновременно, Syncthing создаёт: **filename.sync-conflict**

Это обычная защита данных. Нужно сравнить файлы и оставить нужную версию.

---

### Устройства должны быть онлайн

Syncthing работает только если устройства доступны.

Если устройство выключено, синхронизация произойдёт позже.

---

### Мобильные устройства

На Android есть официальный клиент.

Он работает стабильно, но важно:

- разрешить работу в фоне
- отключить агрессивную экономию батареи

---

# Преимущества Syncthing

### Полная приватность

Нет облака.

Файлы хранятся только на ваших устройствах.

---

### Бесплатно

Нет подписки.

Нет ограничений.

---

### Очень быстрая синхронизация

Если устройства в одной сети — синхронизация происходит почти мгновенно.

---

### Нет лимитов на размер

Можно синхронизировать vault любого размера.

---

# Архитектурно

Syncthing — это **peer-to-peer синхронизация**.

Устройство A  
     ↕  
Syncthing P2P сеть  
     ↕  
Устройство B

Файлы передаются напрямую.

Без центрального сервера.

---

# Когда лучше выбрать другой способ

Syncthing может быть неудобен, если:

- используете **iPhone / iPad**
- устройства редко бывают онлайн одновременно
- хотите простую облачную синхронизацию

В таких случаях лучше:

- [[webdav|WebDAV]]
- [[s3|S3]]
- [[obsidian-sync|Obsidian Sync]]

# FAQ — Частые вопросы

### Поддерживается ли iOS?

Официального клиента нет.

Есть сторонние решения, но они нестабильны.

Поэтому для iPhone лучше использовать:

- [[webdav|WebDAV]]
- [[s3|S3]]
- [[obsidian-sync|Obsidian Sync]]

---

### Нужно ли открывать порты?

Обычно нет.

Syncthing умеет автоматически обходить NAT.

---

### Безопасна ли синхронизация?

Да.

Syncthing использует:

- TLS
- end-to-end encryption

Данные передаются только между вашими устройствами.

---

### Можно ли синхронизировать несколько vault?

Да.

Каждый vault добавляется как отдельная папка.

---

### Можно ли использовать Syncthing вместе с Git?

Да.

Некоторые пользователи используют:

Syncthing → синхронизация устройств  
Git → история изменений
